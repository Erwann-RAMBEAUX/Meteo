A client-side web application that provides real-time weather forecasts, built using the Open-Meteo API and Chart.js.

> **Note:** This project was developed as part of the 2nd-year curriculum for the "BUT Informatique" (University Diploma in Technology - Computer Science).



[Image of weather app interface]


## üéØ Key Features

* **Real-time Weather Data:** Displays current weather conditions (temperature, wind, precipitation) for a searched location.
* **Geocoding Search:** Utilizes a geocoding API to let users easily find and select any city worldwide.
* **Dynamic Temperature Charts:** Renders interactive graphs using **Chart.js** to visualize temperature evolution over time.
* **User Registration:** Includes a functional sign-up form with client-side validation for all fields.

## üõ†Ô∏è Technologies Used

This project is built entirely with front-end technologies:

* **HTML5:** For the core structure of the application.
* **CSS3:** For all styling and layout.
* **JavaScript (ES6+):** For application logic, API calls (`fetch`), and DOM manipulation.
* **[Open-Meteo API](https://open-meteo.com/):** Used as the free data source for all weather and geocoding information.
* **[Chart.js](https://www.chartjs.org/):** Implemented to create responsive and animated data charts.

## üöÄ Running the Project

This is a static, client-side-only application. No server or build process is required.

1.  Clone this repository to your local machine.
2.  Navigate to the project directory.
3.  Open the **`index.html`** file directly in your web browser (e.g., Chrome, Firefox, Safari).
